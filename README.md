# Projet Base de Données - Tifosi

## Description
Création d'une base de données pour un restaurant de street-food italien.

## Contenu
- schema_tifosi.sql : création de la base et des tables
- data_tifosi.sql : insertion des données de test
- tests_tifosi.sql : requêtes SQL de vérification

## SGBD utilisé
MySQL / MariaDB

## Installation
1. Exécuter schema_tifosi.sql
2. Exécuter data_tifosi.sql
3. Exécuter tests_tifosi.sql
